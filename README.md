# Project Aplikasi Sederhana: List Agenda
# Deskripsi
Aplikasi List Agenda adalah sebuah aplikasi web sederhana yang digunakan untuk membantu pengguna dalam mengelola daftar agenda atau kegiatan secara terstruktur dab bisa digunakan untuk suatu perusahaan. Aplikasi ini dibuat untuk mempermudah pengguna agar tidak lupa terhadap jadwal atau kegiatan penting yang akan dilakukan. Di dalam aplikasi ini, Aplikasi List Agenda juga dilengkapi dengan tampilan yang sederhana dan dapat dipahami, sehingga dapat digunakan dengan mudah. Secara keseluruhan, aplikasi List Agenda dirancang sebagai solusi sederhana untuk membantu dalam mengatur agenda harian ataupun dalam jangka waktu yang lama.
